Variables in C++

C++ provides strong type safety with comprehensive support for various data types. Variables must be declared and can be initialized at the same time.

Fundamental Data Types

  • int: Integer values
  • float/double: Floating-point numbers
  • char: Character type
  • bool: Boolean values (true/false)
  • auto: Type deduction (C++11)
