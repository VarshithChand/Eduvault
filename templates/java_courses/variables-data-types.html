Variables & Data Types

Understand Java variables, primitive data types, and type conversion

Understanding Variables

A variable is a named location in memory that stores a value. In Java, you must declare a variable before using it. Each variable has a type, a name, and a value.

Primitive Data Types

Java has 8 primitive data types:

  • byte: 8-bit integer (-128 to 127)
  • short: 16-bit integer (-32,768 to 32,767)
  • int: 32-bit integer (most common for integers)
  • long: 64-bit integer (suffix 'L')
  • float: 32-bit floating-point (suffix 'f')
  • double: 64-bit floating-point (most common for decimals)
  • boolean: true or false
  • char: 16-bit Unicode character

Variable Declaration and Initialization

// Declaration
int age;

// Declaration and initialization
String name = "John";
double salary = 50000.50;
boolean isActive = true;

Type Conversion

Converting data from one type to another can be implicit (widening) or explicit (narrowing).

  • Widening Conversion: From smaller to larger type (automatic)
  • Narrowing Conversion: From larger to smaller type (requires casting)
