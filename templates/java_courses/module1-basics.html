Java Basics & Setup

Learn about Java fundamentals, JDK setup, and write your first program. Understand variables, data types, and basic syntax.

1. Java Environment Setup

Java requires the JDK (Java Development Kit) to compile and run programs. Learn how to set up your environment.

Installing JDK:

// Step 1: Download JDK from oracle.com
// Step 2: Install JDK on your system
// Step 3: Set JAVA_HOME environment variable
// Step 4: Verify installation
java -version
javac -version

Java Naming Conventions:

  • ✓ Class names: PascalCase (MyClass, HelloWorld)
  • ✓ Method names: camelCase (myMethod, printMessage)
  • ✓ Variable names: camelCase (age, userName)
  • ✓ Constants: UPPER_CASE (MAX_VALUE, PI)
  • ✗ Cannot start with numbers or special characters
  • ✗ Cannot use Java keywords (class, public, static, etc.)

2. Your First Java Program

Learn the structure of a Java program and write "Hello World".

Example 1: Hello World Program

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

// Output: Hello, World!

Example 2: Program Structure Explained

public class MyProgram {  // Class declaration (PascalCase)
    public static void main(String[] args) {  // Main method (entry point)
        // Your code goes here
        System.out.println("Java is awesome!");
    }
}

3. Variables and Data Types

Java has eight primitive data types. Learn how to declare and use variables.

Example 1: Primitive Data Types

// Integer types
byte age = 25;           // 8-bit
short population = 30000;  // 16-bit
int salary = 50000;      // 32-bit
long nationalGDP = 2000000000L;  // 64-bit (L suffix)

// Floating point types
float price = 19.99f;    // 32-bit (f suffix)
double pi = 3.14159;     // 64-bit

// Boolean and Character
boolean isStudent = true;
char grade = 'A';

System.out.println("Age: " + age);
System.out.println("Salary: " + salary);

Example 2: String Type (Reference Type)

// Strings are reference types, not primitive
String name = "Alice";
String message = "Hello, " + name;

System.out.println(message);  // Output: Hello, Alice

// String methods
int length = name.length();      // 5
String upper = name.toUpperCase();  // ALICE
String lower = name.toLowerCase();  // alice
boolean hasA = name.contains("li");  // true

Example 3: Type Conversion

// Implicit conversion (widening)
int integer = 100;
double decimal = integer;  // Automatically converted to 100.0

// Explicit conversion (narrowing)
double value = 99.99;
int rounded = (int) value;  // Explicitly cast to 99

// String conversion
String numString = "42";
int number = Integer.parseInt(numString);
double floatValue = Double.parseDouble("3.14");

System.out.println(number);    // 42
System.out.println(floatValue);  // 3.14

4. Input and Output

Learn how to read user input and display output in Java.

Example 1: Output with System.out

// Print with line break
System.out.println("Hello");  // Includes newline

// Print without line break
System.out.print("Hello ");   // No newline
System.out.print("World");    // Output: Hello World

// Formatted output
String name = "Bob";
int age = 25;
System.out.printf("Name: %s, Age: %d%n", name, age);
// Output: Name: Bob, Age: 25

Example 2: Reading User Input

import java.util.Scanner;

public class InputExample {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.print("Enter your name: ");
        String name = sc.nextLine();
        
        System.out.print("Enter your age: ");
        int age = sc.nextInt();
        
        System.out.println("Name: " + name + ", Age: " + age);
        
        sc.close();
    }
}
