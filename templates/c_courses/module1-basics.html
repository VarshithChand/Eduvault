C Language Fundamentals

Learn C basics, compilation, variables, data types, and write your first C program.

1. Setting Up C Environment

C requires a compiler to translate code into machine language. Learn about compilers like GCC.

Installing GCC (GNU Compiler Collection):

// Windows: Download MinGW or use TDM-GCC
// Linux: sudo apt-get install build-essential
// macOS: Install Xcode Command Line Tools

// Verify installation
gcc --version
// Output: gcc (GCC) 9.3.0

C Naming Conventions:

  • ✓ Variable names: lowercase_with_underscores (my_age, user_name)
  • ✓ Constants: UPPER_CASE (MAX_SIZE, PI)
  • ✓ Function names: lowercase_with_underscores (print_message)
  • ✓ Must start with letter or underscore
  • ✗ Cannot start with numbers
  • ✗ Cannot use C keywords (int, if, while, etc.)

2. Your First C Program

Learn the structure of a C program and create "Hello World".

Example 1: Hello World Program

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

// Output: Hello, World!

Example 2: Program Structure

#include <stdio.h>  // Include standard I/O library

int main() {  // Main function (entry point)
    // Your code here
    printf("C Programming\n");
    return 0;  // Return 0 for successful execution
}

3. Data Types and Variables

C has several data types including integers, floats, characters, and more.

Example 1: Basic Data Types

#include <stdio.h>

int main() {
    int age = 25;              // Integer
    float height = 5.9;        // Single precision float
    double weight = 65.5;      // Double precision float
    char grade = 'A';          // Single character
    
    printf("Age: %d\n", age);
    printf("Height: %.1f\n", height);
    printf("Weight: %.2f\n", weight);
    printf("Grade: %c\n", grade);
    
    return 0;
}

Example 2: Strings in C

#include <stdio.h>
#include <string.h>

int main() {
    char name[50] = "Alice";  // Character array (string)
    
    printf("Name: %s\n", name);
    printf("Length: %lu\n", strlen(name));
    
    char combined[100];
    strcpy(combined, "Hello, ");
    strcat(combined, name);
    printf("%s\n", combined);
    
    return 0;
}

Example 3: Type Sizes

#include <stdio.h>

int main() {
    printf("Size of int: %lu bytes\n", sizeof(int));
    printf("Size of float: %lu bytes\n", sizeof(float));
    printf("Size of double: %lu bytes\n", sizeof(double));
    printf("Size of char: %lu bytes\n", sizeof(char));
    
    return 0;
}

4. Input and Output

Learn to read input from user and display output in C.

Example 1: Output with printf()

#include <stdio.h>

int main() {
    int age = 25;
    float salary = 50000.50;
    char grade = 'A';
    
    printf("Age: %d\n", age);           // %d for integer
    printf("Salary: %.2f\n", salary);   // %.2f for float (2 decimals)
    printf("Grade: %c\n", grade);       // %c for character
    printf("Multiple: %d, %f, %c\n", age, salary, grade);
    
    return 0;
}

Example 2: Input with scanf()

#include <stdio.h>

int main() {
    int age;
    char name[50];
    float height;
    
    printf("Enter your name: ");
    scanf("%s", name);      // Read string
    
    printf("Enter your age: ");
    scanf("%d", &age);      // Read integer (& for address)
    
    printf("Enter your height: ");
    scanf("%f", &height);   // Read float
    
    printf("\nName: %s\n", name);
    printf("Age: %d\n", age);
    printf("Height: %.2f\n", height);
    
    return 0;
}
