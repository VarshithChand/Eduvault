Loops

Iterate through data using for and while. Learn loop control (break, continue), nested loops, and loop-else.

1. For loops & the range() function

For loops iterate over sequences (lists, tuples, strings, ranges, etc.). Use range() to generate numeric sequences.

Example 1: Basic for loop

# Iterate over a list
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(f"I like {fruit}")
# Output:
# I like apple
# I like banana
# I like cherry

Example 2: Using range()

# range(stop)
for i in range(5):
    print(i)  # 0 1 2 3 4

# range(start, stop)
for i in range(2, 6):
    print(i)  # 2 3 4 5

# range(start, stop, step)
for i in range(0, 10, 2):
    print(i)  # 0 2 4 6 8

Example 3: Iterating with index - enumerate()

names = ["Alice", "Bob", "Charlie"]
for idx, name in enumerate(names, start=1):
    print(idx, name)
# Output:
# 1 Alice
# 2 Bob
# 3 Charlie

Example 4: Iterating over dictionaries

student = {"name": "Asha", "grade": 92, "id": 101}
# keys
for key in student:
    print(key)

# key/value
for key, val in student.items():
    print(key, "->", val)
# Output:
# name -> Asha
# grade -> 92
# id -> 101

2. While loops & conditions

While loops repeat as long as a condition is true. Be careful to change loop variables to avoid infinite loops.

Example 1: Basic while loop

count = 0
while count < 5:
    print("count is", count)
    count += 1
# Output:
# count is 0
# count is 1
# count is 2
# count is 3
# count is 4

Example 2: User input loop (safe form)

# Keep asking until user types 'quit'
# (In real program use input(); here is pseudocode)
cmd = ""
while cmd != "quit":
    cmd = input("Enter command (type 'quit' to exit): ").strip().lower()
    if cmd:
        print("You typed:", cmd)

Example 3: Using else with while

n = 3
while n > 0:
    print("n =", n)
    n -= 1
else:
    print("Loop finished normally")
# Output:
# n = 3
# n = 2
# n = 1
# Loop finished normally

3. Break, Continue & Loop Control

break exits the loop immediately. continue skips to the next iteration. Use them to control flow inside loops.

Example 1: break example

numbers = [2, 4, 6, 7, 8]
for n in numbers:
    if n % 2 != 0:
        print("Found odd:", n)
        break
# Output:
# Found odd: 7

Example 2: continue example

for i in range(6):
    if i % 2 == 0:
        continue  # skip even numbers
    print(i)
# Output:
# 1
# 3
# 5

Example 3: for/else

target = 5
for i in range(1, 6):
    if i == target:
        print("Found", target)
        break
else:
    # runs only if loop didn't hit break
    print("Not found")
# Output:
# Found 5

4. Nested loops

Loops inside loops are useful for matrices, combinations, and multi-dimensional data. Watch complexity — nested loops can be slow.

Example 1: Multiplication table (nested for)

for i in range(1, 4):
    for j in range(1, 6):
        print(f"{i} x {j} = {i*j}")
    print("---")
# Output: table for 1..3 rows and 1..5 columns

Example 2: Flattening a nested list

matrix = [[1,2,3], [4,5,6], [7,8,9]]
flat = []
for row in matrix:
    for val in row:
        flat.append(val)
print(flat)  # [1,2,3,4,5,6,7,8,9]

Example 3: Combinations (avoid duplicates)

items = ['a','b','c']
pairs = []
for i in range(len(items)):
    for j in range(i+1, len(items)):
        pairs.append((items[i], items[j]))
print(pairs)  # [('a','b'), ('a','c'), ('b','c')]

5. Practical patterns & tips

  • Use enumerate() when you need the index.
  • Prefer for loops over while when iterating a known sequence.
  • Use list comprehensions for simple transformation/filtering (fast and compact).
  • To avoid infinite loops, always make sure loop condition changes inside while.
  • For heavy nested loops, consider algorithm improvements or libraries (numpy, itertools).
