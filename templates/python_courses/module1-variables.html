Variables & Data Types

Learn about integers, floats, strings, and booleans. Understand type conversion and variable naming conventions.

1. Variable Assignment and Naming

Variables are containers for storing data values. In Python, you don't need to declare variable types explicitly.

Example 1: Basic Variable Assignment

# Assigning values to variables
name = "Alice"
age = 25
height = 5.6
is_student = True

print(name)        # Output: Alice
print(age)         # Output: 25
print(height)      # Output: 5.6
print(is_student)  # Output: True

Example 2: Multiple Assignment

# Assign multiple variables in one line
x, y, z = 10, 20, 30
print(x, y, z)  # Output: 10 20 30

# Assign same value to multiple variables
a = b = c = 100
print(a, b, c)  # Output: 100 100 100

Variable Naming Rules:

  • ✓ Must start with a letter or underscore (_)
  • ✓ Can contain letters, numbers, and underscores
  • ✓ Case-sensitive (age, Age, and AGE are different)
  • ✗ Cannot start with a number
  • ✗ Cannot contain spaces or special characters (!@#$%)
  • ✗ Cannot be Python keywords (if, for, while, etc.)

Example 3: Valid and Invalid Variable Names

# Valid variable names
student_name = "John"
_age = 30
name2 = "Bob"
MY_CONSTANT = 100

# Invalid variable names (will cause errors)
# 2name = "Error"        # Cannot start with number
# student-name = "Error"  # Cannot use hyphen
# class = "Error"         # Cannot use keywords

2. Basic Data Types

Python has four basic built-in data types: integers, floats, strings, and booleans.

Example 1: Integer (int)

# Integers are whole numbers
count = 42
negative_num = -17
large_number = 1000000

print(type(count))  # Output: <class 'int'>

# Arithmetic with integers
sum_result = 10 + 5      # 15
difference = 20 - 8      # 12
product = 6 * 7          # 42
quotient = 15 // 3       # 5 (integer division)

Example 2: Float (float)

# Floats are decimal numbers
price = 19.99
temperature = -5.5
scientific = 3.14e2  # 314.0 (scientific notation)

print(type(price))  # Output: <class 'float'>

# Arithmetic with floats
total = 10.5 + 4.3   # 14.8
result = 7.0 / 2.0   # 3.5

Example 3: String (str)

# Strings are text data
message = "Hello, World!"
single_quotes = 'Python is awesome'
multiline = """This is a
multiline string"""

print(type(message))  # Output: <class 'str'>

# String operations
greeting = "Hello" + " " + "World"  # Concatenation
repeated = "Ha" * 3                  # HaHaHa
length = len(message)                # 13

Example 4: Boolean (bool)

# Booleans represent True or False
is_active = True
has_permission = False

print(type(is_active))  # Output: <class 'bool'>

# Boolean operations
result1 = True and False   # False
result2 = True or False    # True
result3 = not True         # False

# Comparison results are booleans
is_greater = 10 > 5        # True
is_equal = 5 == 5          # True

3. Type Casting and Conversion

Convert between different data types using type casting functions.

Example 1: Converting to Integer

# Convert to integer using int()
float_num = 7.8
string_num = "42"

int_from_float = int(float_num)  # 7 (truncates decimal)
int_from_string = int(string_num)  # 42

print(int_from_float)   # Output: 7
print(int_from_string)  # Output: 42

# Boolean to integer
bool_to_int = int(True)   # 1
bool_to_int2 = int(False)  # 0

Example 2: Converting to Float

# Convert to float using float()
integer_num = 10
string_float = "3.14"

float_from_int = float(integer_num)     # 10.0
float_from_string = float(string_float)  # 3.14

print(float_from_int)     # Output: 10.0
print(float_from_string)  # Output: 3.14

Example 3: Converting to String

# Convert to string using str()
number = 100
float_value = 3.14159
bool_value = True

str_from_int = str(number)        # "100"
str_from_float = str(float_value)  # "3.14159"
str_from_bool = str(bool_value)    # "True"

# Useful for concatenation
age = 25
message = "I am " + str(age) + " years old"
print(message)  # Output: I am 25 years old

Example 4: Converting to Boolean

# Convert to boolean using bool()
# Empty/zero values are False, non-empty/non-zero are True

bool(0)         # False
bool(1)         # True
bool("")        # False (empty string)
bool("Hello")   # True
bool([])        # False (empty list)
bool([1, 2])    # True

# Practical example
user_input = "yes"
is_confirmed = bool(user_input)  # True

4. Print Statements and Formatting

Display output and format strings in various ways.

Example 1: Basic Print

# Simple print statements
print("Hello, World!")
print(42)
print(3.14)

# Print multiple items
name = "Alice"
age = 25
print("Name:", name, "Age:", age)
# Output: Name: Alice Age: 25

# Print with separator
print("A", "B", "C", sep="-")  # Output: A-B-C

Example 2: F-strings (Modern Way)

# F-strings - most readable and modern
name = "Bob"
age = 30
height = 5.9

# Basic f-string
print(f"My name is {name}")
# Output: My name is Bob

# Multiple variables
print(f"{name} is {age} years old and {height} feet tall")
# Output: Bob is 30 years old and 5.9 feet tall

# Expressions inside f-strings
x = 10
y = 5
print(f"The sum of {x} and {y} is {x + y}")
# Output: The sum of 10 and 5 is 15

Example 3: Format Method

# Using .format() method
name = "Charlie"
score = 95

message = "Student {} scored {} points".format(name, score)
print(message)
# Output: Student Charlie scored 95 points

# With positional arguments
text = "{0} likes {1} and {2}".format("Alice", "Python", "Coffee")
print(text)
# Output: Alice likes Python and Coffee

# With named arguments
info = "{name} is {age} years old".format(name="David", age=28)
print(info)
# Output: David is 28 years old

Example 4: Formatting Numbers

# Format decimal places
pi = 3.14159265359

print(f"{pi:.2f}")   # 3.14 (2 decimal places)
print(f"{pi:.4f}")   # 3.1416 (4 decimal places)

# Format as percentage
ratio = 0.75
print(f"{ratio:.1%}")  # 75.0%

# Format large numbers with commas
big_number = 1000000
print(f"{big_number:,}")  # 1,000,000

# Padding and alignment
num = 42
print(f"{num:05}")   # 00042 (pad with zeros)
print(f"{num:>10}")  # "        42" (right align)
print(f"{num:<10}")  # "42        " (left align)
