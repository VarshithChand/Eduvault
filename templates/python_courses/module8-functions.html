Functions

Create reusable code blocks with functions. Understand parameters, return values, and scope.

1. Function Definition and Calling

Functions are reusable blocks of code that perform specific tasks. Define with 'def' and call by name.

Example 1: Basic Function Definition

# Define a simple function
def greet():
    print("Hello, World!")

# Call the function
greet()  # Output: Hello, World!

# Function with multiple statements
def display_info():
    print("Welcome to Python functions!")
    print("Functions help organize code.")
    print("They make code reusable.")

display_info()

# Function that performs calculation
def calculate_square():
    number = 5
    square = number ** 2
    print(f"Square of {number} is {square}")

calculate_square()  # Output: Square of 5 is 25

Example 2: Function Naming and Best Practices

# Function names should be descriptive
def calculate_area():
    length = 10
    width = 5
    area = length * width
    print(f"Area: {area}")

calculate_area()  # Output: Area: 50

# Functions can be called multiple times
def say_hello():
    print("Hello!")

say_hello()
say_hello()
say_hello()

# Functions can call other functions
def morning_routine():
    say_hello()
    print("Time for breakfast!")

morning_routine()

# Output:
# Hello!
# Time for breakfast!

# Docstrings for documentation
def greet_user(name):
    """This function greets a user by name."""
    print(f"Hello, {name}!")

greet_user("Alice")  # Output: Hello, Alice!

Example 3: Function Execution Flow

# Code execution jumps to function when called
print("Before function call")

def my_function():
    print("Inside the function")
    print("This executes when function is called")

print("After function definition")
my_function()  # Function call
print("After function call")

# Output:
# Before function call
# After function definition
# Inside the function
# This executes when function is called
# After function call

# Functions are objects
def add_numbers():
    return 5 + 3

result = add_numbers()
print(f"Result: {result}")  # Result: 8

# Assign function to variable
calculator = add_numbers
print(f"Calculator result: {calculator()}")  # Calculator result: 8

2. Parameters and Arguments

Parameters are variables in function definition. Arguments are values passed when calling the function.

Example 1: Positional Parameters

# Function with parameters
def greet_person(name, age):
    print(f"Hello, {name}! You are {age} years old.")

# Call with positional arguments
greet_person("Alice", 25)  # Output: Hello, Alice! You are 25 years old.
greet_person("Bob", 30)    # Output: Hello, Bob! You are 30 years old.

# Multiple parameters
def calculate_rectangle_area(length, width):
    area = length * width
    print(f"Area: {area}")

calculate_rectangle_area(10, 5)  # Area: 50
calculate_rectangle_area(7, 3)   # Area: 21

# Order matters with positional arguments
def divide(a, b):
    result = a / b
    print(f"{a} divided by {b} is {result}")

divide(10, 2)  # 10 divided by 2 is 5.0
divide(2, 10)  # 2 divided by 10 is 0.2

Example 2: Default Parameters and Keyword Arguments

# Default parameter values
def greet_with_default(name, greeting="Hello"):
    print(f"{greeting}, {name}!")

greet_with_default("Alice")          # Hello, Alice!
greet_with_default("Bob", "Hi")      # Hi, Bob!

# Keyword arguments
def create_profile(name, age, city):
    print(f"Name: {name}, Age: {age}, City: {city}")

create_profile(age=25, name="Charlie", city="NYC")
# Output: Name: Charlie, Age: 25, City: NYC

# Mix positional and keyword (positional first)
create_profile("Diana", city="LA", age=28)
# Output: Name: Diana, Age: 28, City: LA

# Default and keyword combined
def send_email(to, subject="No Subject", body=""):
    print(f"To: {to}")
    print(f"Subject: {subject}")
    print(f"Body: {body}")
    print("---")

send_email("user@example.com")
send_email("admin@test.com", subject="Urgent", body="Check server")

Example 3: Variable-Length Arguments (*args and **kwargs)

# *args for variable positional arguments
def sum_all(*numbers):
    total = 0
    for num in numbers:
        total += num
    return total

print(sum_all(1, 2, 3))      # 6
print(sum_all(10, 20, 30, 40))  # 100
print(sum_all())             # 0

# **kwargs for variable keyword arguments
def print_info(**info):
    for key, value in info.items():
        print(f"{key}: {value}")

print_info(name="Alice", age=25, city="NYC")
# Output:
# name: Alice
# age: 25
# city: NYC

# Combine all parameter types
def flexible_function(required, *args, default="default", **kwargs):
    print(f"Required: {required}")
    print(f"Args: {args}")
    print(f"Default: {default}")
    print(f"Kwargs: {kwargs}")

flexible_function("test", 1, 2, 3, extra="value")
# Output:
# Required: test
# Args: (1, 2, 3)
# Default: default
# Kwargs: {'extra': 'value'}

3. Return Statements

Functions can return values using the return statement. Return ends function execution.

Example 1: Returning Values

# Function that returns a value
def add_numbers(a, b):
    return a + b

result = add_numbers(5, 3)
print(f"Sum: {result}")  # Sum: 8

# Use return value in expressions
total = add_numbers(10, 20) + add_numbers(1, 2)
print(f"Total: {total}")  # Total: 33

# Return different types
def get_user_info():
    return "Alice", 25, "Engineer"

name, age, job = get_user_info()
print(f"{name} is {age} years old and works as {job}")

# Return None implicitly (no return statement)
def print_message():
    print("This function doesn't return anything")

result = print_message()
print(f"Return value: {result}")  # Return value: None

Example 2: Multiple Return Points

# Return based on conditions
def check_number(num):
    if num > 0:
        return "Positive"
    elif num < 0:
        return "Negative"
    else:
        return "Zero"

print(check_number(5))   # Positive
print(check_number(-3))  # Negative
print(check_number(0))   # Zero

# Early return
def validate_age(age):
    if age < 0:
        return "Invalid age"
    if age < 18:
        return "Minor"
    return "Adult"

print(validate_age(25))  # Adult
print(validate_age(15))  # Minor
print(validate_age(-5))  # Invalid age

# Return multiple values as tuple
def get_min_max(numbers):
    return min(numbers), max(numbers)

minimum, maximum = get_min_max([3, 1, 7, 2, 9])
print(f"Min: {minimum}, Max: {maximum}")  # Min: 1, Max: 9

Example 3: Return vs Print

# Print vs Return
def print_square(num):
    print(num ** 2)  # Prints the result

def return_square(num):
    return num ** 2  # Returns the result

# Using print function
print_square(4)  # Output: 16
result = print_square(4)  # Output: 16
print(f"Returned value: {result}")  # Returned value: None

# Using return function
result = return_square(4)
print(f"Square: {result}")  # Square: 16

# Practical example: Calculator
def calculator(a, b, operation):
    if operation == "add":
        return a + b
    elif operation == "subtract":
        return a - b
    elif operation == "multiply":
        return a * b
    elif operation == "divide":
        if b != 0:
            return a / b
        else:
            return "Cannot divide by zero"
    else:
        return "Invalid operation"

print(calculator(10, 5, "add"))       # 15
print(calculator(10, 5, "divide"))    # 2.0
print(calculator(10, 0, "divide"))    # Cannot divide by zero

4. Local and Global Scope

Variables have scope - where they can be accessed. Local variables are inside functions, global outside.

Example 1: Local Variables

# Local variables exist only inside functions
def my_function():
    local_var = "I'm local"
    print(local_var)

my_function()  # Output: I'm local

# This would cause an error:
# print(local_var)  # NameError: name 'local_var' is not defined

# Different functions can have same local variable names
def function1():
    x = 10
    print(f"function1 x: {x}")

def function2():
    x = 20
    print(f"function2 x: {x}")

function1()  # function1 x: 10
function2()  # function2 x: 20

# Parameters are also local
def greet(name):
    greeting = f"Hello, {name}!"
    print(greeting)

greet("Alice")  # Hello, Alice!
# print(name)  # NameError
# print(greeting)  # NameError

Example 2: Global Variables

# Global variables can be accessed everywhere
global_var = "I'm global"

def access_global():
    print(global_var)  # Can read global variable

access_global()  # Output: I'm global

# Modifying global variables inside functions
count = 0

def increment():
    global count  # Declare global to modify
    count += 1
    print(f"Count: {count}")

increment()  # Count: 1
increment()  # Count: 2
print(f"Global count: {count}")  # Global count: 2

# Avoid modifying globals when possible
def better_increment():
    # Use return instead of global
    return count + 1

count = better_increment()
print(f"New count: {count}")  # New count: 3

Example 3: Scope Hierarchy and nonlocal

# Nested functions and nonlocal
def outer_function():
    outer_var = "outer"

    def inner_function():
        nonlocal outer_var  # Access outer function's variable
        outer_var = "modified by inner"
        inner_var = "inner"
        print(f"Inner: {outer_var}, {inner_var}")

    inner_function()
    print(f"Outer after inner: {outer_var}")

outer_function()
# Output:
# Inner: modified by inner, inner
# Outer after inner: modified by inner

# Global in nested functions
global_counter = 0

def counter_function():
    def increment():
        global global_counter
        global_counter += 1
        return global_counter
    return increment

counter = counter_function()
print(counter())  # 1
print(counter())  # 2

# Built-in scope (avoid shadowing)
# Don't do: len = 5  # This would shadow built-in len()

# LEGB Rule: Local, Enclosing, Global, Built-in
def demonstrate_scope():
    var = "local"
    print(var)  # local

    def inner():
        print(var)  # enclosing (if no local var)
    
    inner()

demonstrate_scope()

  • celsius_to_fahrenheit(celsius)
  • fahrenheit_to_celsius(fahrenheit)
  • celsius_to_kelvin(celsius)
  • kelvin_to_celsius(kelvin)

  • Support add, subtract, multiply, divide
  • Handle division by zero
  • Return the result or an error message
  • Use if-elif for different operations

Create functions to calculate statistics on a list of numbers:

  • calculate_mean(numbers)
  • calculate_median(numbers)
  • calculate_mode(numbers)
  • find_min_max(numbers) - return both min and max

Exercise 4: Password Validator

Write a function that validates passwords with these criteria:

  • At least 8 characters long
  • Contains at least one uppercase letter
  • Contains at least one lowercase letter
  • Contains at least one digit
  • Return True/False and list of failed criteria
